Why the fix kept failing

The old loop proved that Lasso could dispatch a click. It did not prove that X accepted the click.

Weak proof

element.click() fired, so the code reported success.

Problem: X could ignore the event and leave the menu open.

Strong proof

The menu closed, the tweet changed, and the feedback button was clicked.

That proves the target app, not just our script, processed the action.

The pattern to remember

Browser extensions run content scripts in an isolated world. That world can touch the DOM, but page apps can still behave differently from a real user click. When a site ignores synthetic clicks, use a tiny main-world bridge and verify a site-owned side effect.
